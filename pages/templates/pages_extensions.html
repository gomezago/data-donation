Exploring your Data

If you have downloaded a copy of your data and you want to understand its content this is the place to start. Here you can find brief explanations on how to manually inspect it.

  • Clue Data: The data you download from the Clue app (file ending in .cluedata) is in a format called JSON which stands for JavaScript Object Notation. JSON files can be opened using a text editor (TextEdit on Mac or Notepad on Windows) or with Google Chrome and Mozilla Firefox, by dragging and dropping the file into your browser window. This file contains the following information:
    • Environment: Information about your device, device configuration (e.g., language and timezone), and app version.
    • Data: Information you provide to the app by manually inputting your symptoms. Every field represents an input to the app (e.g., weight, period, mood, exercise, digestion) and contains information about the input (e.g., value, type) as well as the date (day, month, year) in which you reported it.
    • Settings: Information about the way you configured the app when you first used it. This includes the average length of your cycle, and your preferred fields (e.g., yes exercise, no birth control).

  • Google Voice and Audio Data: The data you download from the Google Takeout (file starting with takeout) is in a ZIP format. ZIP files contain compressed files and folders and can be opened by double-clicking on the file, in Mac, or right-clicking on the file and selecting “Extract All” in Windows. The file contains the following:
    • Archive Browser File: An HTML file, that you can open with any web browser, containing information about your download including the time, date and total size.
    • "My Activity" Folder: A folder containing the voice recordings for every query you have made to your voice assistant in mp3 format (listen by double-clicking), together with a “My Activity” File, an HTML file where you can see the time, date and transcript of each query.
